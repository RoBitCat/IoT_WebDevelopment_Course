Why Flex?

Flex is a useful feature that allows seemless intregration between different users across multiple devices. Be it mobile, desktop or tablet.
It makes a responesive website. Meaning that it respones the content to the size of the browser without a need of different template.

Here we have list of 10 text using the flex attributes. Which reacts to the change in website width.

This attribute is affected by 2 main elements in class defined as .flex in CSS:

  • display: flex; -> Enable the flex function.
  • flex-wrap: wrap; -> Changes the flex-wrap style to wrap. Meaning anything that is overflowing (going off screen) will get pushed into a new line
You may try this by resizing the browser window width to see how it affects the flex element

This Flexbox is a <div> with class defined as .container giving as <div class="container">.
This .container class has a similar element to the text from example 1.1. but builds up on it.
A new line .container > div is added in CSS file. The > symbol is called the child combinator. It makes sure that any <div> element nested inside of <div class="container"> is selected at any depth.

In the CSS file the .container file has the following .container > div {background-color: rgb(0, 46, 61); padding: 1rem; margin: 0.5; flex-grow: 1; text-align: center;}

  • Padding: seen as the empty area around the text
  • Margin: the white area around the flex items with the texts
  • Flex-grow set to 1 to allow the flex items to "grow" to the max avaliable space
  • Text-align Set to center to center the text
